# DeepSleep Mobile

A React Native Web application for sleep tracking and mental health assessment.

## 🚀 Live Demo

**Production URL:** https://deepsleep-mobile.vercel.app

## 📱 Features

- **Sleep Tracking**: Log sleep quality, duration, and feelings
- **Progress Analytics**: Interactive charts and statistics
- **Mental Health Assessment**: Integrated PHQ-9 and GAD-7 assessments
- **Education**: Sleep science articles and tips
- **User Profiles**: Secure authentication and data management

## 🏗️ Tech Stack

- **Frontend**: React Native Web, Expo, TypeScript
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **UI**: React Navigation, Custom Components
- **Deployment**: Vercel

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/nic-lobby/deepsleep-mobile.git
cd deepsleep-mobile

# Install dependencies
npm install

# Start development
npm run web
```

## 📦 Deployment

### Web (Vercel)
1. Import repository to Vercel
2. Set environment variables:
   - `EXPO_PUBLIC_SUPABASE_URL`
   - `EXPO_PUBLIC_SUPABASE_ANON_KEY`
3. Deploy

### Mobile (App Stores)
```bash
expo build:ios
expo build:android
```
